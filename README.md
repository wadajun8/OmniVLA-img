# [OmniVLA](https://github.com/NHirose/OmniVLA) Docker Image
## Docker Support
- We provide a Makefile to simplify the environment setup.

### Initial Setup: Run this command once to build the Docker image and prepare dependencies.

> Note: This process involves building a large Docker image and downloading dependencies. It may take a significant amount of time depending on your environment.
```
make setup
```
### Usage: To start the container and enter the development environment, simply run:
```
make
```
